Project overview
I want to create a custom cricket score recording app. however, the twist is that we play indoor cricket which has its own special rules. 
these are the things i want the app to be able to do
users provide the app with a full list of players. this is the eligible universe of players that can play. 
users at the start of each match select 2 teams and a captain based off the eligible universe of players
each match is set up of 2 to 3 games. first 2 games each player bowls 1 over of 6 fixed balls. last over no balls and wides add balls to the over.
when the over starts, user should have the option of selecting the bowler based on the available team members. 
once bowler is chosen, the user should have options of numbers 1-12 to enter as the score for each ball
the score per innings should be cumalatively tracked.
